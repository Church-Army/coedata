# National statistics socio-economic classification levels

A reference table that describes the NS-SEC classifications of
socio-economic status that have been compounded to create the measures
provided by the coe_data package. These measures are used in the ONS
dataset 'TS062'.

## Usage

``` r
ns_sec_descriptions
```

## Format

### `ns_sec_descriptions`

A tibble with 15 rows and 3 columns:

- stat_name:

  Statistic name as used in coe_data datasets

- sub_category_code:

  Numerical code corresonding to NS-SEC sub-category

- sub_category_description:

  Description of sub-category

## Source

https://www.ons.gov.uk/datasets/TS062/editions/2021/versions/1
