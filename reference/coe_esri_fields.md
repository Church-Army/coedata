# View fields that can be passed to the `where` argument of [`coe_churches()`](https://church-army.github.io/coedata/reference/coe_churches.md)

View fields that can be passed to the `where` argument of
[`coe_churches()`](https://church-army.github.io/coedata/reference/coe_churches.md)

## Usage

``` r
coe_esri_fields()
```

## Value

A character vector of field names

## Examples

``` r
coe_esri_fields()
#>  [1] "FID"        "Church_Cod" "ChurchName" "ACNY_id"    "ACNY_url"  
#>  [6] "Church_Her" "LBGrade"    "ParLocNa"   "ParLegNa"   "ParishID"  
#> [11] "BenefNa"    "BenefID"    "DeanNa"     "DeanID"     "ArchdNa"   
#> [16] "ArchdID"    "DioNa"      "DioID"     
```
